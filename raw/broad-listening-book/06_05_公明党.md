# 第6章 国政選挙でのブロードリスニングの利用

文責: tokoroten

## 公明党：与党として取り組んだブロードリスニング

### 若者離れに直面した与党のブロードリスニング

公明党は支持層の高齢化という構造的課題を抱えていた。支持団体である創価学会の会員高齢化に伴い、高齢の支持者の自然減が進む一方で若年層の支持は伸びず、衆院選の比例票はピーク時の900万票弱から2024年には600万票を割り込んでいた[^komei_vote]。若年層の支持獲得は党勢拡大に向けた喫緊の課題だった。

こうした背景の中で、2025年3月17日、公明党は「政策立案アンケートWe Connect（ウィーコネクト）」を開始した。参議院選挙（7月20日投開票）の重点政策に若者や現役世代の「生の声」を反映させることを目的としたプロジェクトで、アンケート期間は3月17日から6月26日までの約3か月にわたった。以降、本節では「We Connect」と略記する。

当時の公明党は自民党と連立を組む与党であり、本章の他の事例（チームみらい・維新・国民民主）がいずれも野党だったのに対し、政策を実現できる立場でブロードリスニングに取り組んだ点が異なる。

なお本節は公明党への直接取材は行っておらず、特設サイト・公式YouTube・公開資料等の公開情報に基づいて構成している。

### We Connectの意見集約から政策決定までのダブルダイヤモンド型のプロセス

We Connectの政策立案プロセスは、特設サイトで明示的に公開されていた。単なるアンケートの一回収集ではなく、2回のラウンドのサイクルとして設計されていた点が特徴だ。

![We Connectの政策立案の流れ](images/06_05_WeConnect_workflow.jpg)[^komei_flow]

まず3月17日にアンケート第一弾を開始し、4月中下旬からGWにかけて寄せられた声を集約して党内で議論する。5月のGW明けにはアンケート第二弾を実施し、第一弾で得られた声をもとに政策案をさらに議論する。第一弾のアンケートは第二弾開始後も引き続き回答可能で、アンケートと並行して議員による個別調査も行われた。こうして重ねた議論をもとに6月に党の政策調査会が政策を決定し、参院選公示前に重点政策として発表するという流れだ。

このプロセスは、デザイン思考で知られる「ダブルダイヤモンド」モデルに似た構造を持っている。まず22の政策分野にわたって広く意見を集め（発散）、AIで重点分野に絞り込み（収束）、再び意見を集め（発散）、最終的に政策として決定する（収束）という2回の発散と収束のサイクルだ。収束フェーズでAIを活用することで、大量の意見を人手だけに頼らず集約できる設計になっていた。

集まった声の分析にAIを活用することは、特設サイトに明記されていた。

> 「アンケートでいただいた声やご要望を、**AIなどのデジタル技術を活用し**、どのような方（属性）から、どのような声やご要望が多いかなど、多角的に分析します。若者・現役世代などのニーズをとらえ、政策立案に繋げていきます。」

単に意見を集めるだけでなく、AIによって誰がどんな声を上げているかを属性ごとに把握し、政策に結びつけることが最初から意図されていた。与党としての公明党がAI活用を政策立案プロセスに組み込むと宣言したことは、ブロードリスニングの主流化を示す一つの指標といえる。

We Connectのアンケート開始の同日には、公明党の公式YouTubeチャンネルにプロジェクトの趣旨が解説されたショート動画が投稿された[^komei1]。

> 「政治家だけで議論すると足元の課題だけに偏りがち」

> 「国民の皆さま、とりわけ若い世代の皆さまから、直接『こういうことを実現してほしい』『こういうことに困っている』（といった）お声を寄せていただいて、そのことを公明党の政策のど真ん中に据えて、必ず実現してまいります」

> 「皆さまから頂いたお声を（特設サイトやSNSで）共有させていただきたいと思っています。そのうえで『ここが足りていない』『もっとこういうこともすべきだ』（といった）さらに多くの声をいただきながら、最終的にそれをまとめたものを与党・責任政党・公明党として必ず実現していきたいと決意しています」

この動画が示すように、We Connectは「市民の声を党が集約し、党が政策に翻訳して実現する」という設計だ。市民同士の直接の熟議ではなく、与党としての政策実現力を前提に、党が仲介者として中心に立つモデルである。

### 収集の仕組み：Google FormとQRコードによる草の根の拡散

意見収集の窓口は、専用サイト「weconnect.jp」に設けられた入力フォームだ。実装はGoogle Formを埋め込む形で行われており、高度な独自システムではなく、既存のツールを活用したシンプルな構成だった。党員でなくとも誰でも回答でき、メールアドレスも収集されなかった。

このフォームへの誘導は、複数の経路で行われた。公明党の公式SNS（X、YouTube、Instagramなど）への投稿、機関紙や「公明ハンドブック2025」などの紙媒体に掲載されたQRコード、そして全国の地方議員が街頭演説や地域活動を通じてQRコードを直接案内する草の根の呼びかけだ。SNSの役割はあくまでフォームへの誘導と中間報告の場に限定されており、他党のようにXのハッシュタグからSNS投稿をスクレイピングするような手法は採られていなかった。

### 第一弾アンケートの結果、103,519件の意見から何が見えたか？

2025年3月17日に開始された第一弾アンケートの設問は3部構成だった。まず性別・年齢・居住地域・職業・婚姻状況・子どもの有無という6項目の属性情報を収集し、次に「物価高対策」「雇用・労働問題」「教育・子育て支援」「住宅」「孤独・孤立対策」「外国人との共生」など22の政策分野から関心のあるものを最大3つ選択させる。最後に「今、課題に感じていることや、政治への意見、政策への要望・アイデアなど」を自由記述で入力する形だ。属性と関心分野をセットで取得する構造により、「30代・子育て中・雇用に関心」といったクロス分析が可能になっていた。

4月30日時点で91,216件の回答が集まり、最終的に6月24日時点で103,519件に達した。集まった声は、AIを活用して意見をグループ分けして分析されたとされる。ただし、どのようなアルゴリズムで解析したのかは非公開であるため、ここでは分析の詳細には踏み込まず、公開された情報からわかる範囲で結果を紹介する。

We Connectが始まった2025年3月は、消費者物価指数（生鮮食品を除く総合）が前年同月比+3.2%と高止まりしており[^cpi]、米価格が前年比ほぼ2倍に達していた時期だ[^rice]。「令和の米騒動」の渦中で、政府は前例のない備蓄米放出の入札をWe Connect開始直前に行っていた。

こうした時代背景を反映して、集まった声のワードクラウドでは「備蓄米」「物価高」「税金」「減税」「政策」「議員」「世帯」「給付金」「支援」といった語が大きく浮かび上がる。

![We Connectに寄せられた声のワードクラウド](images/06_05_WeConnect_wordcloud.png)[^komei3]

政策課題の選択結果（複数回答可）では、「物価高対策・景気対策」が約65,000件と断トツで、「年金・医療・介護などの社会保障」（約50,000件）、「教育・子育て支援・少子化対策」（約25,000件）が続いた。

一方で注目すべきは、「政治とカネの問題などの政治改革」が下位に沈んだことだ。公明党は結党以来「クリーンな政治」を党の看板に掲げ、2024年の政治資金規正法改正でもいわゆる連座制の強化やパーティー券購入者の公開基準引き下げを主導した政党である[^komei_reform]。

にもかかわらず、10万件超の有権者の声は物価高や社会保障といった生活に直結する課題に集中し、党が60年間掲げてきた看板への関心は相対的に低かった。ブロードリスニングは、党の訴求軸と有権者のニーズとのギャップを可視化する機能も果たしたことになる。

![政策課題の選択件数](images/06_05_WeConnect_policy_issues.png)[^komei3]

年代別のクロス集計では、「物価高対策・景気対策」は10代（47.5%）から70代以上（71.3%）まで全世代で最多だった。一方で関心の差が出たのは「年金・医療・介護」で、50代（53.0%）・60代（69.2%）・70代以上（71.0%）と高齢層ほど関心が高い反面、10代（23.0%）では相対的に低い。「教育・子育て支援」は30代（48.5%）・40代（43.5%）でピークを迎え、子育て世代の実態と一致している。

![年代×政策課題のクロス集計](images/06_05_WeConnect_age_policy.png)[^komei3]

世代と関心分野をクロスさせることで、「若年層が本当に求めている政策は何か」を可視化し、若年層の支持獲得につなげるための手がかりを得ることが、このアンケート設計の狙いだったと考えられる。

### 第二弾アンケート：収束のフェーズへ

第二弾アンケートは、第一弾とは設問の構造が大きく異なっていた。属性情報は性別と年齢の2項目に絞られ、第一弾で収集した居住地域・職業・婚姻状況・子どもの有無は省略された。第一弾で属性分析が完了しているため、第二弾では回答のハードルを下げることを優先した設計と考えられる。

設問の中心も変わった。第一弾が22の政策分野から「関心のあるもの」を選ぶ発散的な問いだったのに対し、第二弾は第一弾の声をもとに党が練り上げた6つの具体的な政策案を「推進してほしい政策案を選択してください」とビジュアルカード形式で提示し、回答者に評価させる収束的な問いになっていた。6つの政策案は以下の通りだ。

- 【物価高支援】米や食料品の負担軽減策・給付・ガソリン代引き下げ・電気・ガス代補助など
- 【社会保障】介護休職中の所得補償・介護施設の利用料の助成・住宅支援など
- 【教育】既卒者も対象の返済免除制度の拡充・給付型奨学金の拡大など
- 【子育て】子どもの医療費無償・公教育の質向上・子どもを取り巻く環境改善など
- 【雇用・労働】ライフステージに合わせた働き方の選択肢拡大・週休3日制など
- 【雇用・労働】保育士･介護士の賃金大幅増・職場環境改善など

![We Connect第二弾アンケートの政策選択画面](images/06_05_WeConnect_survey2.png)[^komei3]

各政策案はキャッチコピー付きのカードで視覚的に提示されており、22分野のプレーンなチェックボックスだった第一弾とは対照的だ。第二弾には23,147件の回答が寄せられ、中でも「家計支援、今すぐ実行を。」は約8割の支持を集めた。

### 政策への反映

期間終了後、公明党は「若者の声が参院選重点政策になりました」として、集まった声がどのように参院選の重点政策[^komei_manifesto]に結びついたかをWe Connectのサイトで公開した[^komei3]。主な反映政策として以下が挙げられている。

| テーマ | 主な反映政策 |
|---|---|
| 家計支援 | 所得税減税（基礎控除引き上げ）、生活応援給付（18歳以下・低所得世帯に4万円等）、夏の電気・ガス料金軽減、自動車ユーザー減税・ガソリン暫定税率廃止、コメの増産・価格安定 |
| 介護 | エッセンシャルワーカーの所得向上（公定価格への適切な反映）、ワーキングケアラー支援（相談体制強化・介護休業取得促進）、医療介護DX（ICT化・オンライン診療） |
| 教育 | 奨学金減税（返済額の所得控除・減額返還制度の年収制限緩和・代理返還制度の拡充） |
| 子育て | 学校給食無償化、高校授業料無償化、大学授業料支援、妊娠・出産無償化、こども誰でも通園制度、不登校支援、35人学級推進、住宅手当の創設、育児休業取得促進、放課後児童対策の拡充 |
| 働き方 | 最低賃金を2020年代に全国平均1500円へ、同一労働同一賃金（「正規／非正規」概念の廃止）、リスキリング支援、106万・130万円の壁見直し、高齢者就業支援 |
| 保育士・介護士 | 給与を全産業平均（月額38.6万円）まで引き上げ |

参院選の重点政策の周知にもAIが活用された。重点政策の概要はAIプレゼンテーションツール「Gamma」で作成・公開された[^komei_gamma]。政策案は作れば終わりというわけではなく、さまざまな情報チャネルで周知していく必要がある。そのため、それぞれのチャネルに合わせた形に政策案を変換することが重要だ。Gammaは、政策案をわかりやすいビジュアル資料に変換するのに役立ったと考えられる。

![Gammaで作成された公明党「2025重点政策」の概要ページ](images/06_05_WeConnect_gamma.png)[^komei_gamma]

加えて、公明新聞電子版には「AIと見つけよう！あなたのための公明マニフェスト」と題したAI政策検索機能が搭載された[^komei_search]。これは自由文で質問を入力すると、AIが60項目の重点政策の中から関心に最も近い公約を探し出して解説するもので、内部的にはベクトル検索（テキストの意味的な類似性に基づく検索）が使われていると推測される。また、個別の政策については、「なぜこの政策が必要なのかAIに聞く」というボタンが設置されており、AIから政策の背景を聞くことができるようになっていた。

意見の収集から分析、政策立案、そして有権者への周知まで、プロセスの各段階でAIを組み込んだ点は、We Connectの大きな特徴といえる。

### 党組織とブロードリスニングの組み合わせ

本章の他の事例と比較すると、公明党のWe Connectはいくつかの点で異なる性格を持っている。

まず規模について。チームみらい（8,559件）や国民民主党（6万件）と比べ、第一弾・第二弾合計で約12万7千件という回答数は本章で最大だ。ただしこの規模は、全国に広がる公明党の地方議員ネットワークが草の根でQRコードを配布したことによるところが大きいと考えられる。SNS上の不特定多数から意見を収集した維新（30万件収集）とは収集構造が異なる。

次に収集手法について。維新はSNS投稿のスクレイピング、チームみらいはAIインタビューを経由したプルリクエスト、国民民主はオンライン・オフライン混合の複数チャネルと各党が工夫を凝らしたのに対し、公明党はGoogle Formというシンプルな手段を選んだ。技術的な複雑さではなく、既存の組織力とツールを組み合わせることで大規模な意見収集を実現した点が特徴的だ。

公明党はかねてから「小さな声を聴く力」を党の強みとして掲げてきた。We Connectはその伝統的な姿勢を、デジタルツールを用いて現代的な形にアップデートする試みだったといえる。

しかし結果として、7月の参院選で公明党は改選14議席に対し8議席の獲得にとどまり、比例票も521万票（前回比−97万票）に落ち込んだ。党の総括文書は現状を「党存亡の危機」と位置づけ、敗因として「40〜50代現役世代、10〜30代若年層で支持伸び悩み」を挙げている[^komei_soukatsu]。We Connectで約12万7千件の声を集めたにもかかわらず、若年層の支持獲得には至らなかった。

選挙の敗因はブロードリスニングの手法だけで説明できるものではないが、We Connectには構造的な限界があったと考えられる。党組織の草の根ネットワークでQRコードを配布するという手法は、12万件超の回答を集める原動力になった一方で、届く範囲を党の既存の支持基盤に限定してしまった可能性がある。本来獲得すべき「公明党と接点のない若者層」の声がどれだけ含まれていたかは不明だ。ブロードリスニングの成否は、ツールの洗練度だけでなく、誰にリーチできるかという構造によっても左右されることを、この事例は示唆している。

一方で、総括文書には今後の党改革として、「リアルタイム情報共有が可能なAIエージェント」の導入や、「演説及びSNS発信の質向上に向けた強化プログラム検討」が明記されている[^komei_soukatsu]。党がデジタル分野への人員・予算のシフトを打ち出し、SNSでの発信力強化を正面から課題として認めた点は、We Connectで若者との接点づくりを試みた延長線上にある。また、We Connectから生まれた奨学金減税は参院選の重点政策に採用された。We Connectは選挙結果という点では成果に結びつかなかったが、AI活用を政策立案プロセスに組み込む方向性と、有権者の声から生まれた政策を公約に反映させるという成果は残した。
